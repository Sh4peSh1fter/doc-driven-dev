# Architecture Decision Records

Each ADR captures one significant decision: its context, what was decided, the alternatives, and the consequences. ADRs are numbered `NNN-title.md` and are never deleted — superseded ones are marked, not removed.

## Index

| ADR | Title | Status |
|:----|:------|:-------|
| [000](./000-record-architecture-decisions.md) | Record architecture decisions | Accepted |

Create new ADRs from the [ADR template](../../_templates/_template.adr.md). See the [ADR process](../../11-documentation/_standards/README.md#adrs-architecture-decision-records) for when and how.
