# Archive

Frozen content: superseded docs, deprecated designs, and historical records kept for reference. Nothing here is authoritative. Move a doc here (rather than deleting it) when it is replaced, and set its `status` to `archived`.

This folder is excluded from validation and from the published site (it is listed in `validator.skipDirs` in `docs/.docmeta.json`).
