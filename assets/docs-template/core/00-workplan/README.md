# Workplan

Roadmaps, backlogs, and task tracking. This section is for planning what to build next and recording the status of in-flight work. It is the home for trackers (`_tasks/*.tasks.md`), audit findings, and roadmap notes.

**Audience:** anyone planning or coordinating work — engineers, leads, and product.

## Contents

Add roadmap and backlog pages here as the project grows. Use the [tracker template](../_templates/_template.tracker.md) for task lists and the [doc template](../_templates/_doc-template.md) for narrative roadmap pages.
